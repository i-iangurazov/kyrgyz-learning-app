# Curriculum Sequencing Rules

This document defines how lessons must be logically sequenced. It protects the product from becoming a collection of disconnected exercises, random vocabulary, or unvalidated grammar notes.

## Core Rule

Every lesson should teach one practical communication or reading outcome through a controlled path:

Vocabulary -> Dialogue/Text -> Breakdown -> Grammar -> Practice -> Review

The learner should see language in context before being asked to analyze or produce it.

## Lesson Scope

Rules:

- Each lesson has one main communication goal.
- Each lesson has one main grammar point.
- Each lesson has limited new vocabulary.
- K0 lessons may replace a grammar point with one reading, alphabet, or pronunciation focus.
- Lessons should feel completable in 8-12 minutes.

Recommended vocabulary load:

- K0: 3-6 new items.
- K1: 5-8 new items.
- K2: 6-10 new items.
- K3: 8-12 new items.
- K4: 10-14 new items.
- K5: 12-16 new items.

## Vocabulary Before Grammar

Rules:

- Do not introduce grammar before required vocabulary unless necessary.
- Vocabulary should prepare the learner for the dialogue/text.
- Every vocabulary item should appear in context.
- Phrases count as vocabulary when they are memorized as chunks.
- Later lessons should reuse earlier vocabulary intentionally.

Allowed exception:

- K0 alphabet or pronunciation lessons may introduce letters/sounds before full vocabulary, but examples still need clear context.

## Dialogue/Text Before Explanation

Rules:

- Every grammar point must be used in the dialogue or text before explanation.
- Dialogue/text should mostly use known or target language.
- New unknown items must be intentional and supported.
- Avoid direct word-for-word translation from English or Russian.
- Keep dialogue and reading length level-appropriate.

## Grammar Sequencing

Rules:

- One main grammar point per lesson.
- Grammar must be immediately useful for the lesson goal.
- Grammar explanations should start simple.
- Track-specific explanations should be added where useful.
- Common mistakes should be added when they help practice or review.
- Grammar must be validated by a Kyrgyz methodist/linguist before production release.

Do not:

- Add grammar only because it is academically interesting.
- Stack multiple suffix rules in one beginner lesson.
- Present sample explanations as final authority.

## Practice Sequencing

Rules:

- Practice must reinforce the current lesson, not random content.
- Exercise difficulty should move from recognition to controlled production.
- Wrong answers should teach and feed review.
- Supported exercise types should be chosen for the lesson goal.

Recommended order:

1. Recognition: multiple choice, match pairs, listening choice later.
2. Controlled recall: fill blank.
3. Construction: sentence builder.
4. Correction: error correction.
5. Open production later: short answer, speaking, roleplay.

## Review Sequencing

Rules:

- Review must include missed items.
- Review should summarize what the learner can now do.
- Corrected missed items should remain available in review history.
- Practice tab review should focus first on needs-review items.
- Future flashcards should be generated from lesson content and mistakes.

## Flashcard Sequencing

Rules:

- Flashcards must come from lesson content, approved knowledge records, or learner mistakes.
- New flashcards should not introduce untaught grammar.
- Common mistake cards should link to the source exercise or grammar point.
- SRS should prioritize weak and due material, not random content.
- Use a proven SRS implementation such as FSRS rather than custom scheduling.

## Level Sequencing

K0:

- Alphabet basics.
- Special Kyrgyz letters: ө, ү, ң.
- Greetings and short polite phrases.
- Micro-reading.
- Confidence with Kyrgyz text on mobile.

K1:

- Everyday survival communication.
- Greetings.
- Introductions.
- Family.
- Numbers.
- Food.
- Cafe.
- Shop.
- Bazaar.
- Taxi.
- Address.

K2:

- Expanded daily interactions.
- Directions, time, schedules, preferences, weather, and simple narratives.

K3:

- Connected language.
- Routines, events, opinions, short messages, announcements, and adapted cultural texts.

K4:

- Intermediate functional communication.
- Experiences, comparisons, service problems, short informational texts, and public language.

K5:

- More independent communication.
- Literature, essays, argumentation, stylistics, public speaking, and longer reading/listening.

## Literature And Reading Sequencing

Rules:

- Advanced literature comes later through graded reading.
- K0/K1 should not rely on long authentic texts.
- K2 can use adapted folk stories and simple biographies.
- K3 can use adapted school-literature themes and culture/history texts.
- K4 can use semi-authentic public language.
- K5 can approach original literature and stylistics.
- All reading content requires source and rights notes.

## AI Roleplay Sequencing

Rules:

- AI roleplay comes after vocabulary, dialogue/text, grammar, and practice.
- AI must use approved vocabulary and grammar for the lesson/level.
- AI roleplay should not introduce new authoritative grammar explanations.
- Scenarios should reuse the lesson situation and goal.

## Sequencing Acceptance Checklist

- [ ] Lesson has one primary outcome.
- [ ] New vocabulary is limited.
- [ ] Every vocabulary item appears in context.
- [ ] Dialogue/text appears before grammar.
- [ ] Grammar appears in the dialogue/text.
- [ ] Practice tests current or previously taught content.
- [ ] Missed items feed review.
- [ ] Flashcards can be generated from linked content later.
- [ ] Later lessons reuse earlier vocabulary or grammar.
- [ ] Source, rights, and methodist status are recorded internally.
