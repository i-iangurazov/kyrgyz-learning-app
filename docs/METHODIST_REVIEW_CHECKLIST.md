# Methodist Review Checklist

Use this checklist before any lesson is treated as final learner-facing content.

## Lesson Metadata

- [ ] Lesson ID is stable.
- [ ] Level, unit, and lesson order are correct.
- [ ] Lesson goal is clear and observable.
- [ ] Content status is correct: demo, in review, approved, or rejected.
- [ ] Reviewer name and review date are recorded when applicable.

## Vocabulary Accuracy

- [ ] Kyrgyz spelling is correct.
- [ ] Meaning is accurate.
- [ ] Register and politeness are appropriate.
- [ ] Transliteration or pronunciation notes are approved if present.
- [ ] Vocabulary load fits the level.
- [ ] Vocabulary supports the lesson goal.

## Grammar Accuracy

- [ ] The lesson has no more than one main grammar point.
- [ ] Explanation is accurate.
- [ ] Examples are natural Kyrgyz.
- [ ] Examples show the target pattern clearly.
- [ ] Common mistakes are accurate and useful.
- [ ] Track-specific explanations are accurate.

## Naturalness Of Kyrgyz

- [ ] Dialogue sounds natural.
- [ ] Short text sounds natural.
- [ ] Phrases match the intended context.
- [ ] Word choice is appropriate for the learner's situation.
- [ ] No unnatural direct translation from English or Russian remains.

## Translation Quality

- [ ] English translation is accurate.
- [ ] Russian translation is accurate.
- [ ] Translation preserves meaning, not just word order.
- [ ] Any cultural or register nuance is noted.
- [ ] KY-only explanation is level-appropriate where present.

## Level Appropriateness

- [ ] Vocabulary fits the level.
- [ ] Grammar fits the level.
- [ ] Dialogue length fits the level.
- [ ] Text length fits the level.
- [ ] Exercises fit the level.
- [ ] Speaking and AI roleplay prompts fit the level.

## Cultural Appropriateness

- [ ] Scenario is realistic.
- [ ] Names, places, and examples are respectful.
- [ ] Cultural assumptions are checked.
- [ ] Sensitive topics are avoided unless intentionally designed and reviewed.

## Exercise Correctness

- [ ] Every closed exercise has a correct answer.
- [ ] Distractors are fair.
- [ ] Feedback is accurate.
- [ ] Exercises test taught or reviewed material.
- [ ] Exercise instructions are clear in each supported track.

## Audio Readiness

- [ ] Audio is required for this item or explicitly deferred.
- [ ] Native speaker recording is available where required.
- [ ] Audio matches written content.
- [ ] Audio quality is acceptable.
- [ ] Pronunciation coaching text is approved.

## AI Roleplay Safety

- [ ] Scenario is linked to lesson content.
- [ ] Allowed vocabulary is defined.
- [ ] Allowed grammar is defined.
- [ ] Correction style is appropriate.
- [ ] AI is instructed not to invent authoritative grammar explanations.
- [ ] Off-level fallback is defined.
- [ ] Review tags are defined where possible.

## Copyright Safety

- [ ] Dialogue is original.
- [ ] Text is original.
- [ ] Exercises are original.
- [ ] No textbook passage or exercise set is copied.
- [ ] Any licensed source material is documented.
- [ ] External sources are used only for methodology, sequencing, or validation unless licensed.

## Final Approval Status

- [ ] Approved for demo only.
- [ ] Approved for internal testing.
- [ ] Approved for learner-facing MVP.
- [ ] Rejected with revision notes.
- [ ] Requires additional linguist review.
- [ ] Requires additional methodist review.
